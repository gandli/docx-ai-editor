## 快速参考

```bash
# 开发时
bun test --watch              # 监听模式

# 提交前
bun test:run                  # 运行所有测试
bun test:coverage             # 生成覆盖率

# 调试
bun test -t "测试名称"         # 运行特定测试
bun test:e2e --headed         # E2E 有头模式
bun test:e2e --debug          # E2E 调试模式

# CI/CD
bun test:unit                 # 单元测试
bun test:integration          # 集成测试
bun test:e2e                  # E2E 测试
```
